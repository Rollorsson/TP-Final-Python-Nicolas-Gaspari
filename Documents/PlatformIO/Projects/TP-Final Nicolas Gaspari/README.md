# Predicción de Demanda Eléctrica en La Pampa (Invierno 2026)

Este proyecto utiliza datos históricos para predecir la demanda eléctrica diaria en invierno en la provincia de La Pampa, Argentina, aplicando un modelo de Machine Learning (Random Forest Regressor) en Python.

## Descripción
- Se analizan factores como el uso de calefacción eléctrica, consumo residencial nocturno y variaciones de temperatura extrema.
- El flujo de trabajo incluye exploración de datos, ingeniería de variables, entrenamiento y evaluación de modelos, y visualización de resultados.
- El notebook está documentado paso a paso y contiene gráficos explicativos para facilitar la interpretación.

## Visualizaciones
- Se muestran gráficos claros y descriptivos para comparar la demanda real con la predicha y analizar el error absoluto por día.
- Cada gráfico incluye leyendas y comentarios para que cualquier persona pueda entender el análisis.

## Colaboración con IA
Este trabajo fue realizado originalmente en inglés y luego mejorado y traducido al español con la ayuda de un agente inteligente en VS Code (GitHub Copilot), quien colaboró en la documentación, visualización y claridad del código para facilitar la exposición y comprensión del proyecto.
